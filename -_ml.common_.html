truncate

angular filter for truncating text.

truncates to length, offset by the length of end, and concatenates with end.

ex:

'abcdefg' | truncate:5 returns 'ab...'
'abcdefg' | truncate:5:'' returns 'abcde'

Parameters

length

Number

 

output length (defaults to 10)

end
